### Cara Menjalankan Program:
    - nyalakan XAMPP
    - buka terminal
    - masuk ke direktori program
    - buat virtual environment dengan perintah `python -m venv env`
    - aktifkan virtual environment dengan perintah `env\Scripts\activate`
    - install requirements dengan perintah `pip install -r requirements.txt`
    - jalankan program dengan perintah `python main.py`
